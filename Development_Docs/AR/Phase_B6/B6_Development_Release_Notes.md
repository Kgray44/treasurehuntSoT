# 0.8.0-b6 development release notes

This is an **experimental NO-GO development build**, not a public Creator Preview or stable release.

It adds a persisted release authority/dashboard, deterministic partitioned replay and bounded soak tools, package/resource/signature hardening, canonical signed release metadata, atomic update rollback primitives, Windows signing/provenance scripts, migration-safe certification/compatibility data, local pinned Rive WASM, Creator/Player/Captain onboarding, metadata-only improvement review, current diagnostics/version reporting, and an assisted per-user NSIS build.

Public use remains blocked by production signing, hosted CI evidence, real locked pilot corpora, hardware/game-impact and long-duration testing, clean-machine install/update/uninstall, external usability/accessibility, independent security/privacy review, and Rare/Microsoft distribution clarification.
