# Community Remix Lineage and Licensing

Every editable copy or fork records source release, source package checksum, root release, upstream tracking mode and immutable attribution snapshot. Forks preserve upstream attribution in order and append the new contributor. Compatibility compares immutable release snapshots: prohibited modification, public-use, commercial, source-link and share-alike obligations block a plan with a typed reason. Mutable Creator defaults never rewrite historical obligations.
