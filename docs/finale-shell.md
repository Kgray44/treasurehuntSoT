# Finale shell

The schema supports `HIDDEN`, `TEASED`, `SEALED`, `REQUIREMENTS_PARTIAL`, `READY`, `UNLOCKED`, and `COMPLETE`, but Phase 2 uses only the first four. Public requirements are generic symbolic counters. Optional stars are marked optional and do not block the main voyage.

No real finale copy, object, dependency, destination, or unlock sequence is present. `READY`, `UNLOCKED`, and `COMPLETE` remain structural future states and must not be activated until a later authorized phase.
