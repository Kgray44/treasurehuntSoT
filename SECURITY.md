# Security reporting

Do not file package passphrases, private Chronicle prose, export files, access
tokens, or credentials in issues, chat transcripts, or pull requests. Report a
suspected disclosure privately to the repository owner with only the minimum
reproduction metadata. Rotate a confirmed credential before requesting any
history rewrite.

Private Chronicle packages use the `.ftprivate` format and must remain outside
the repository, public static paths, build output, and ordinary application
logs. Run the private-content repository, staged-diff, and public-build scans
before sharing a change.
